# Fluent Emoji を Slack に導入する

**📍 Point**
1. シェルスクリプトでアップロードの準備をする
2. Chrome 拡張を使って一括で絵文字を追加する

## シェルスクリプトでアップロードの準備をする

GitHub から Fluent Emoji のリポジトリをダウンロードします。

```
git clone git@github.com:microsoft/fluentui-emoji.git
```

シェルスクリプトでアップロードの準備をします。  
`png` というフォルダを作り、絵文字のファイルをすべて、そのフォルダ内に複製します。


```
find . -name '*.png' -type f | xargs -I {} cp {} png
```


## Chrome 拡張を追加して、絵文字を一括で追加する

[Nertral Face Emoji Tools](https://chrome.google.com/webstore/detail/neutral-face-emoji-tools/anchoacphlfbdomdlomnbbfhcmcdmjej) を Chrome に追加します。

その後、 『ワークスペースのカスタマイズ』を開きます。

<img src="/posts/fluent-emoji-to-slack_1.webp" />

`png` フォルダ内のファイルをドラッグ・アンド・ドロップしたら Slack に一括で絵文字を追加できます。

<img src="/posts/fluent-emoji-to-slack_2.webp" />

## おわりに

素敵な Slack ライフを！
